Работно време на календарите

Промяната на стандартното работно време се прави директно в Acuity. Използвайте блокирането на часове за временни промени.

Как да промените работното време:

  1. Влезте в Acuity (или рестартирайте паролата)
  2. Отидете на Calendar → Business Hours
  3. Променете работните часове за всеки ден
  4. Запазете промените

Алтернативен метод чрез блокове:

Ако нямате достъп до Acuity, можете да използвате блокове за да симулирате промяна на работното време:

  1. За да затворите сутрините: Блокирайте 00:00-09:00 за всеки ден
  2. За да затворите вечерите: Блокирайте 18:00-23:59 за всеки ден
  3. За почивни дни: Блокирайте целия ден 00:00-23:59

Управление на услуги

Услуги: Управлявайте вашите услуги, тяхната продължителност, цени и параметри.
Важно: Редактирането на услуги чрез API изисква специфичен Acuity план. Ако получите грешка при опит за запазване на промени, моля редактирайте услугите директно в Acuity Scheduling панела.
🔧 Заобиколен вариант: Спиране на услуга без директна редакция

За потребители без достъп до Acuity профил: Ако нямате възможност да влезете в Acuity, можете ефективно да "спрете" услуга, като блокирате всички календари на дълъг период.

Важно: Този метод блокира ЦЕЛИЯ календар (всички услуги), не само избраната услуга. Използвайте го само ако искате да спрете приемането на всякакви резервации за този период.

Как работи:

  • Създава блок (unavailable period) за всички календари
  • Блокът предотвратява ВСИЧКИ резервации в календара
  • Ефективно "спира" приемането на нови резервации
  • Можете да изтриете блока, за да възстановите достъпността
Забележка: Избраната услуга се използва само за проследяване. Блоковете ще засегнат всички услуги в календара.

API Ключове

За регенериране на API ключове за по-голяма сигурност:

  1. Влезте в Acuity (използвайте password reset ако е нужно)
  2. Отидете на Settings → Integrations → API
  3. Натиснете "Regenerate API Key"
  4. Обновете Worker secrets с новите стойности
Команди за обновяване на Worker secrets: 
wrangler secret put ACUITY_USER
wrangler secret put ACUITY_KEY
wrangler deploy

СПЕШЕН РЕЖИМ "LOCKDOWN"

Този режим блокира ВСЕ календари за нови резервации, като запазва съществуващите. Използвайте САМО ако подозирате активно хакване на акаунта.

Какво прави Lockdown:

  • ✓ Създава блокове за всички календари за следващите 30 дни
  • ✓ Запазва съществуващите резервации
  • ✓ Предотвратява нови резервации от клиенти
  • ✓ Може да бъде обратим (премахване на блоковете)
